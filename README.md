# Cinematic Bars Mod

A simple and stylish Minecraft mod that adds cinematic black bars to the top and bottom of your screen for a more cinematic viewing experience.

## Features

- Black cinematic bars at the top and bottom of the screen
- Configurable bar height and color
- Client-side only (can be used on any server)
- Lightweight and performance-friendly

## Installation

1. Download the latest JAR file from [Releases](https://github.com/chappcG/cinematic-bars/releases)
2. Install [Minecraft Forge](https://files.minecraftforge.net/) for 1.20.1
3. Place the JAR file in your `.minecraft/mods` folder
4. Launch Minecraft with the Forge profile

## Building from Source

### Prerequisites
- JDK 17 or higher
- Git

### Steps

```bash
git clone https://github.com/chappcG/cinematic-bars.git
cd cinematic-bars
./gradlew build
```

The compiled JAR will be in `build/libs/`
